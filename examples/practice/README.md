# LeRobot 实践示例

这个目录包含了 LeRobot 的快速入门和实践示例。

## 📁 文件说明

- **`quick_start.py`** - 快速入门脚本，演示如何加载和查看数据集
- **`PRACTICE_GUIDE.md`** - 完整的实践指南，包含所有使用说明
- **`LEARNING_STEPS.md`** - 完整学习步骤指南（15个步骤，5个阶段）
- **`CHAT_HISTORY.md`** - 学习对话历史记录，包含重要知识点和流程详解
- **`NEXT_STEPS.md`** - 下一步行动指南
- **`UNDERSTANDING_PIP_INSTALL.md`** - pip install -e . 命令详解

## 🚀 快速开始

### 1. 运行快速入门示例

```bash
# 从项目根目录
python examples/practice/quick_start.py

# 或者进入目录后运行
cd examples/practice
python quick_start.py
```

### 2. 查看实践指南

```bash
cat examples/practice/PRACTICE_GUIDE.md
```

## 📚 更多示例

项目中的其他示例位于：

- `examples/dataset/` - 数据集使用示例
- `examples/training/` - 训练策略示例
- `examples/tutorial/` - 各种策略的详细教程
- `examples/lekiwi/` - LeKiwi 机器人完整工作流

## 💡 建议的学习路径

1. ✅ 先运行 `quick_start.py` 了解基本概念
2. 📖 阅读 `PRACTICE_GUIDE.md` 了解完整工作流
3. 🔍 查看 `examples/dataset/` 中的数据集示例
4. 🎯 尝试 `examples/training/` 中的训练示例
5. 🤖 如果有硬件，尝试录制和训练自己的数据
